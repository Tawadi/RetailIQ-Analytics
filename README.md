# RetailIQ Analytics Platform

## Project Overview
RetailIQ Analytics Platform is an end-to-end retail analytics solution built to simulate a real-world enterprise project.

## Objectives
- Build a normalized SQL database
- Create an ETL pipeline using Python
- Build interactive Power BI dashboards
- Perform advanced SQL analysis
- Apply Machine Learning for demand forecasting
- Integrate AWS S3 for cloud storage

## Tech Stack
- SQL (MySQL)
- Python
- Excel
- Power BI
- Machine Learning (Scikit-learn)
- AWS (S3)
